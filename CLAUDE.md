# HE Feature Visualizer

Interactive viewer for H&E histopathology images with multi-channel immunofluorescence overlays (Lin 2021 CRC Atlas / CRC02 tissue sample).

## Tech Stack

- **Language:** Python 3.13
- **Backend:** FastAPI + Uvicorn
- **Frontend:** Vanilla HTML/JS + OpenSeadragon 4.1.0 (Deep Zoom)
- **Key libs:** tifffile, Pillow, pandas, numpy, zarr, scipy
- **Testing:** pytest

## Project Structure

```
server.py           # FastAPI server — DZI tiles, heatmap/cell data
preprocess.py       # One-time preprocessing: CSV + TIFF → cache/
patchify.py         # Crop WSI into 256×256 patches
server_patches.py   # Lightweight patch viewer server
viewer.html         # Main viewer UI
viewer_patches.html # Patch viewer UI
ome_reader.py       # OME-TIFF utilities
cell_state.py       # Cell type classification logic
tissue_mask.py      # HSV-based tissue detection
pipeline_generate.py
cellvit_backend.py  # CellViT model integration stub
tests/              # pytest test suite (7 modules)
cache/              # Generated by preprocess.py (gitignored)
data/               # Raw data files (gitignored)
processed/          # Generated by patchify.py (gitignored)
```

## Data Flow

```
OME-TIFF + CSV features → preprocess.py → cache/ (parquet + meta.json + tiles/)
                                               ↓
                                          server.py → viewer.html

Alternative (limited hardware):
OME-TIFF + CSV → patchify.py → processed/ → server_patches.py → viewer_patches.html
```

## Dev Commands

```bash
# Setup
pip install -r requirements.txt

# Preprocess (one-time, 5–15 min)
python preprocess.py --features data/crc02-features.csv --image data/CRC02-HE.ome.tif --out cache/

# Run full viewer
python server.py --image data/CRC02-HE.ome.tif --cache cache/ --port 8000
# Open http://127.0.0.1:8000

# Run patch viewer
python patchify.py --image data/CRC02-HE.ome.tif --features-csv data/CRC02.csv --out processed/ --stride 256 --features cell_mask vasculature immune --tissue-min 0.1
python server_patches.py --processed processed/ --port 8000

# Tests
pytest
```

## Key Architectural Notes

- **Two-stage pipeline:** preprocess once, serve many — never re-read raw TIFF at query time
- **DZI tiling:** multi-level tile pyramid for efficient pan/zoom
- **OME-TIFF + zarr backend:** fast region access for multi-channel images
- **Parquet cell database:** indexed features for fast spatial queries
- Cell types: Tumor (Keratin+), Immune (CD45+), Stromal (aSMA+), Vasculature (CD31+)
- Immune subtypes: CD8a, CD68, FOXP3, CD4, CD20
